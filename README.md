# LinkedIn-UI-Clone
“Professional UI/UX design prototype of a LinkedIn-inspired website, featuring sign-up, onboarding, and landing page interfaces, created in Figma for educational purposes.”

💻🎨 LinkedIn UI/UX Prototype

This project is a **UI/UX design prototype of a LinkedIn-inspired website**, created in **Figma** as part of a **college mini-project** for educational purposes. 🎓  

It includes:

✍️ Sign-up screens  
🚀 Onboarding / "Join LinkedIn" screens  
🏠 Landing / home page designs  

✨ Features

🖌 Clean, modern layout inspired by LinkedIn  
📏 Consistent spacing, typography, and colors  
🎨 Designed entirely in Figma as part of college coursework  

🛠 Tools Used

🖥 Figma  
🎯 UI/UX design principles  

🎯 Purpose

This project was developed as a **college mini-project** to practice **UI/UX design skills** and gain experience in creating professional-looking web interfaces.
